## Installing

```bash
git clone --recursive https://github.com/corneliu-rotari/.dotfiles ~/.dotfiles
cd .dotfiles
./install.sh
```

```bash
git clone --recursive git@github.com:corneliu-rotari/.dotfiles.git ~/.dotfiles
cd .dotfiles
./install.sh
```

## Config

```bash
./config.sh
```

### NVIM

Run the next commands

```vim
:MasonInstallAll
```
